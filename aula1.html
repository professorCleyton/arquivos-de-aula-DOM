O que é o DOM?

O DOM (Document Object Model) é a representação de dados dos objetos que compõem a estrutura e o conteúdo de um documento na Web. MDN - Versão pt-BR Documentação Oficial

W3C o que é e qual a importância?

O Consórcio World Wide Web (W3C) é um consórcio internacional no qual organizações filiadas, uma equipe em tempo integral e o público trabalham juntos para desenvolver padrões para a Web. W3C BR W3C BR
